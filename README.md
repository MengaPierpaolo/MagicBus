# TDiary.React
---
A React.js UI to Consume the API data served by .NET Core TDiary Project
---
# System Requirements
1. Node & npm
2. `npm install webpack browser-sync -g`

# Initial setup instructions
1. Ensure the System Requirements are met
2. Clone the rep
3. `npm install`
4. `npm run dev`
5. Browse to index.html
